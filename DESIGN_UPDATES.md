# Espace Parfum - Luxury Design Transformation

## Overview
The e-commerce website has been completely redesigned with a luxury, modern, and minimalist aesthetic inspired by high-end perfume boutiques and Parisian elegance.

## Design System

### Color Palette
- **Primary**: Deep Pink (`#8B3A4A`) - Elegant and sophisticated
- **Accent**: Rose Gold (`#D4956F`) - Luxury and warmth
- **Background**: Cream (`#FDFBF8`) - Soft and premium
- **Text**: Muted Foreground - Refined and readable
- **Soft Blush**: Light pink accents for subtle highlights

### Typography
- **Serif Font**: Cormorant Garamond (light weight) - Elegant titles and headings
- **Sans-serif Font**: Inter (light weight) - Clean body text and UI elements
- **Tracking**: Wide letter spacing for luxury feel
- **Font Weights**: Primarily light (300) and medium (500) for minimalist aesthetic

### Spacing & Layout
- Generous white space throughout
- Breathing room between elements
- Max-width containers for elegant presentation
- Consistent padding and margins

## Key Design Features

### Micro-interactions
- **Smooth transitions** (300-500ms) on all hover states
- **Scale animations** on product images (105-110%)
- **Gradient overlays** that appear on hover
- **Shadow elevation** for depth perception
- **Fade-in animations** for page load

### Responsive Design
- **Mobile-first approach** with breakpoints at md (768px) and lg (1024px)
- **Flexible grid layouts** that adapt to screen size
- **Touch-friendly** button and link sizes
- **Optimized spacing** for mobile devices

### Accessibility
- **Proper contrast ratios** for text readability
- **Semantic HTML** structure
- **Focus states** for keyboard navigation
- **Icon labels** and descriptive text
